# DesignWeekend

A special project created for **Design Weekend 2025** — a design competition challenge in Plovdiv, Bulgaria.  
**Trak-a-trak** is a web platform for a kids’ robotics academy, teaching them how to build  LEGO music-making robots and providing an easy registration page for new participants.

## Additional information

- The project **Trak-a-trak** is an assignment, based on two special key words – **"star"** and **"gaida"** (a traditional bulgarian instrument).
- The project, along with the provided design graphics and other special assignments, was awared **2nd place** in the Design Weekend 2025 competition.
